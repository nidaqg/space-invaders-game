## Description

A simple browser-based game built using javascript and css. Built using the following tutorial: https://youtu.be/3Nz4Yp7Y_uA 

## Table of Contents

* [Usage](#usage)
* [Contributing Guidelines](#contributing)
* [License](#license)

## Usage 
To play the game, use the left and right arrow buttons to move the shooter and use the spacebar to shoot.
